void DisplayNumberOnLcd ( uint32_t data,
uint8_t digits,
gpioConf_t * bcd_gpio,
gpioConf_t * sel_gpio )

Muestra un número en un display multiplexado usando BCD y selección de dígito.

Parámetros
dataNúmero a mostrar (máximo 'digits' dígitos).
digitsCantidad de dígitos a mostrar.
bcd_gpioVector de estructuras gpioConf_t para los pines BCD (4 elementos).
sel_gpioVector de estructuras gpioConf_t para los pines de selección de dígito (tantos como dígitos).

Esta función convierte el número a BCD, selecciona cada dígito del display y envía el valor BCD a los pines correspondientes. Incluye un pulso de latch para asegurar el registro de datos en el CD4543.
